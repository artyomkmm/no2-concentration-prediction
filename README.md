# NO2 Concentration Prediction

This project focuses on predicting NO₂ concentration using air quality sensor data.

The project was completed as part of a two-month Machine Learning program organized by RBK Bank. As a result of successfully completing the program, I was selected for a paid internship as a Data Scientist.

## Project Overview

The main goal of this project is to build a machine learning model that can predict NO₂ concentration based on environmental and sensor-based features.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook / Google Colab

## Goal

To analyze air quality data, preprocess sensor measurements, train machine learning models, and evaluate their performance in predicting NO₂ concentration.
